# UI Help Localization

## RU

Встроенная справка в UI должна:

- быть понятной не только разработчику
- объяснять термины вроде `signal`, `selector`, `quality`, `raw`, `units`
- переключаться по выбранному языку
- по умолчанию открываться на русском

### Правила

- служебные идентификаторы можно оставлять на английском
- объяснение рядом с ними должно быть на выбранном языке
- сложные сущности должны иметь встроенную подсказку прямо в UI

### Текущий минимум

- переключатель языка `ru/en`
- встроенная справка во вкладке `Signals`
- отдельная справка по `Selector block`

## EN

Built-in UI help must:

- be understandable for operators, not only developers
- explain terms such as `signal`, `selector`, `quality`, `raw`, and `units`
- switch with the selected UI language
- default to Russian for the current product baseline

### Rules

- technical identifiers may stay in English
- nearby explanation should follow the selected language
- complex concepts must have inline help directly in the UI
