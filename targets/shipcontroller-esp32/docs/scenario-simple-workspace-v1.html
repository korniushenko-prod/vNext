Инспектор

Настройки только выбранного узла. Без перегруза internals на первом экране.
Timer selected
Что делает
Запускает цикл только когда enable=true. Публикует активность, фазу и оставшееся время.
Основные поля
Режим, ON time, OFF time, stop policy, enable source.
Автосоздастся
owned timer block, published timer outputs, module status bindings.
Advanced
Generated internals, runtime signals и raw blocks открываются только по запросу.

Что меняется этой упрощённой моделью

  • На первом экране больше нет отдельной большой колонки `Authority / Policy`.
  • Режимы `ON/OFF`, `AUTO/MANUAL`, `LOCAL/REMOTE` остаются в компактном верхнем контексте.
  • Пользователь редактирует один выбранный сценарий, а не всю внутреннюю архитектуру сразу.
  • Generated internals и runtime graph уходят в `Advanced`, а не занимают главный экран.
  • Сценарий остаётся центром проекта, но простая логика не обязана сразу превращаться в отдельный sequence.
