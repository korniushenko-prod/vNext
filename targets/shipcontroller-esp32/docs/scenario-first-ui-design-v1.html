Выделенный child sequence

Когда кусок Main дорос до фаз и переходов, он становится отдельным sequence.
sequence runtime

Burning

→
Cooldown
enabled = false
active = false
Почему это лучше
  • Простой пользователь начинает с `Main` и не видит raw blocks.
  • Продвинутый пользователь может выносить логику в child sequences без переделки проекта.
  • Сценарий становится основной единицей мышления, а модули — основным языком сборки.
  • Blocks остаются честным advanced-слоем, а не первым экраном продукта.
