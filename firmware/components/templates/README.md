# Templates Component

Stage 23 adds a typed Template Engine that builds safe runtime bundles from curated templates.

Scope:
- typed template drafts
- runtime catalog integration
- validation and preview
- disabled-by-default bundle generation
- safe apply with rollback

Non-goals in this stage:
- HTTP server
- MQTT
- template persistence/import-export
- template delete/unapply
- certified burner logic
